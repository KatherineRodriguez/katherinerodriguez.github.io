A Little Bit About Myself...

By Katherine Rodriguez
*In my first year in Technology Production I learned to use an application called Adobe Photoshop. I learned the basics of the application and worked on variety of different projects. I have two of my best projects from my first year in the Adobe Photoshop tab. I really enjoy working on the application with the assigned projects from my tech teacher Mr. Pate or just for my YouTube channel for every thumbnail or graphic. In sixth grade I also learned Adobe Dreamweaver to make a website based on tips for pets such as dogs, cats, gerbils, and gold fish.

*In seventh grade I learned the application known as Adobe Illustrator that you can use for the pen tool to illustrate. Only learning the program for one year, The application wasn't very difficult to learn and in my seventh grade year, I leaned more to using it. I created some of my best work that you can check out in the gallery in the Adobe Illustrator tab. 

*For eighth grade, I learned how to create a basic 2D animation on Adobe Animate, I also learned the program Mobirise to create this website, and I learned a speck of the application known as Premiere Pro to edit film. Mobirise wasn't as hard to learn Adobe Dreamweaver which in sixth grade I learned and made my first website. But you can not compare the two, I prefer Mobirise in a heartbeat because it's easier and I don't have to struggle with codes in Adobe Dreamweaver.
